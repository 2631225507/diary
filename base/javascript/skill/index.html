实用的JS技巧

# 一、日期

# 1. 获取各个日期数据

该方法用于检测给出的日期是否有效：

const date = new Date(); //得到日期对象
date.getYear();     // 获得的是当前年份减去1900的数值
date.getFullYear(); // 获取完整的年份(4位)
date.getMonth();    // 获取当前月份(0-11,0代表1月) 所以按照国人的习惯，获得到的月份都+1
date.getDate();     // 获取当前日(1-31)
date.getDay();      // 获取当前星期X(0-6,0代表星期天)
date.getTime();     // 获取当前时间(从1970.1.1开始的毫秒数) 时间戳
date.getHours();    // 获取当前小时数(0-23)
date.getMinutes();  // 获取当前分钟数(0-59)
date.getSeconds();  // 获取当前秒数(0-59)
date.toLocaleDateString(); // 获取当前日期   格式为 2021/3/11
date.toLocaleTimeString(); // 获取当前时间   格式为 上午11:53:36
date .toLocaleString( );   // 获取日期与时间  格式为 2021/3/11上午11:54:35

# 四、数字

# 1. 取整/四舍五入方法汇总

银行家舍入法的规则是“四舍六入五考虑，五后非零就进一，五后为零看奇偶，五前为偶应舍去，五前为奇要进一“ 向上取整：ceil

Math.ceil(11.01); // 12
Math.ceil(11.99); // 12
1
2

向下取整：floor

Math.floor(11.01); // 11
Math.floor(11.99); // 11
1
2

四舍五入：round（保留整数位）

Math.round(11.01); // 11
Math.round(11.99); // 11

# 完美方案
const round = (n, d) => Number(Math.round(n + "e" + d) + "e-" + d)
round(1.005, 2) // 1.01
round(1.555, 2) // 1.56

# 2. 监听页面是否可见（最小化、正常展示）

可以用来节省资源的消耗，一旦用户不看网页，对服务器的轮询、网页动画、正在播放的音频或视频等都是可以暂停的：

document.addEventListener('visibilitychange', () => {
  switch (document.visibilityState) {
    case 'visible':
      console.log('页面可见')
      break
    case 'hidden':
      console.log('页面不可见')
      break
  }
})

const typeCheck = (type) => Object.prototype.toString.call(type).slice(8, -1).toLowerCase()

typeCheck();       // undefined
typeCheck(null);   // null
typeCheck({});     // object
